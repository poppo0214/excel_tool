# 発注数計算ツール 操作マニュアル

## 1. はじめに

### 1.1 本ツールの目的

本ツールは、各課の希望数量をもとに、予算内に収まる発注数量を自動計算するツールです。

希望数量のまま発注することもできますが、予算を超える場合は自動的に数量を調整します。

また、自動計算後に担当者が発注数量を手動で調整することもできます。

---

### 1.2 本ツールでできること

* 商品ごとの発注数量計算
* 課ごとの発注数量計算
* 予算超過時の自動調整
* 予算内での数量増加
* 発注数量の手動調整
* 予算との差額確認

---

## 2. 画面構成

### 2.1 シート構成

本ツールは以下の2シートで構成されています。

| シート名  | 用途        |
| ----- | --------- |
| 基本設定  | 商品名・課名の登録 |
| 発注数計算 | 発注数量の計算   |

---

### 2.2 セル色の意味

| 色  | 意味         |
| -- | ---------- |
| 青色 | 入力セル       |
| 黄色 | 手動調整可能セル   |
| 白色 | 自動計算セル     |
| 赤字 | 直近の変更・反映箇所 |

# 4. 初期設定

## 4.1 始める

### 操作

① Excelファイルを開く

② 「始める」ボタンを押す

### 実行内容

* 基本設定シートを作成
* 発注数計算シートを作成
* 入力欄を作成
* メモを設定

### 注意

「始める」を押すと、

* 基本設定シート
* 発注数計算シート

の内容は初期化されます。

---

# 5. 商品名・課名の登録

## 5.1 商品名の入力

### 操作

① 「基本設定」シートを開く

② B列の商品名欄に入力する

### 入力例

| No | 商品名   |
| -- | ----- |
| 1  | コピー用紙 |
| 2  | ボールペン |
| 3  | 付箋    |

### 入力ルール

* 最大20個
* 1件以上必須
* 上から順番に入力
* 同じ商品名は不可

---

## 5.2 課名の入力

### 操作

① E列の課名欄に入力する

### 入力例

| No | 課名  |
| -- | --- |
| 1  | 総務課 |
| 2  | 営業課 |
| 3  | 技術課 |

### 入力ルール

* 最大20個
* 1件以上必須
* 上から順番に入力
* 同じ課名は不可

---

# 7. 希望数の入力

## 7.1 予算入力

### 操作

① 発注数計算シートを開く

② B2セルに予算を入力する

## 7.2 単価入力

### 操作

* 正の整数
* 単位は円

## 8.1 予算チェック・自動計算

### 操作

以下のメッセージが表示されます。

「予算内で増やしますか？」

### いいえ

希望数どおりの数量を採用します。

---

## 8.3 希望金額が予算超過の場合

数量を比率調整して予算内に収めます。

# 9. 手動調整

## 9.1 課別発注数を変更する場合

### 操作

① 黄色セルの課別発注数を変更

② 「手動調整・再計算」を押す

### 結果

発注数合計が再計算されます。

---

## 9.2 発注数合計を変更する場合

### 操作

① 黄色セルの発注数合計を変更

② 「手動調整・再計算」を押す

### 結果

課別発注数が希望数比率で再配分されます。

---

### 例

変更前

| 総務課 | 営業課 | 技術課 | 発注数合計 |
| --- | --- | --- | ----- |
| 10  | 20  | 30  | 60    |

発注数合計を72へ変更

↓

変更後

| 総務課 | 営業課 | 技術課 | 発注数合計 |
| --- | --- | --- | ----- |
| 12  | 24  | 36  | 72    |

---

## 9.3 赤字表示について

手動変更したセルは赤字になります。

再計算により反映されたセルも赤字になります。

次回変更時に前回の赤字は黒色へ戻ります。

---

# 10. 状態の確認

## 10.1 状態欄

状態欄には以下が表示されます。

| 項目     | 内容     |
| ------ | ------ |
| 予算     | 入力した予算 |
| 発注金額   | 発注総額   |
| 予算との差額 | 予算残額   |

---

### 差額の見方

| 値    | 意味     |
| ---- | ------ |
| プラス  | 予算残あり  |
| 0    | 予算ぴったり |
| マイナス | 予算超過   |

# 11. エラー一覧

## 11.1 発注数計算表作成時

| エラー内容                   | 原因          | 対処方法        |
| ----------------------- | ----------- | ----------- |
| 基本設定シートに商品名・課名を入力してください | 商品名または課名未入力 | 商品名・課名を入力する |
| 商品名が重複しています             | 同じ商品名がある    | 重複を解消する     |
| 課名が重複しています              | 同じ課名がある     | 重複を解消する     |
| 商品名の途中に空欄があります          | 入力順が飛んでいる   | 上から詰めて入力する  |
| 課名の途中に空欄があります           | 入力順が飛んでいる   | 上から詰めて入力する  |

---

## 11.2 予算チェック・自動計算時

| エラー内容                     | 原因     | 対処方法        |
| ------------------------- | ------ | ----------- |
| 予算は正の整数で入力してください          | 予算が不正  | 正の整数で入力する   |
| ○○の単価は正の整数で入力してください       | 単価が不正  | 正の整数で入力する   |
| ○○×○○の希望数は0以上の整数で入力してください | 希望数が不正 | 0以上の整数で入力する |

---

## 11.3 手動調整・再計算時

| エラー内容                  | 原因      | 対処方法           |
| ---------------------- | ------- | -------------- |
| 変更された発注数セルがありません       | 変更していない | セルを変更してから再計算する |
| ○○の発注数は0以上の整数で入力してください | 発注数が不正  | 0以上の整数で入力する    |

---

## 11.4 状態確認時

| エラー内容           | 原因       | 対処方法    |
| --------------- | -------- | ------- |
| 発注金額が予算を超過しています | 発注額が予算超過 | 発注数を減らす |

---

# 12. 最終確認チェックリスト

計算完了後、以下を確認してください。

□ 商品名・課名は正しい

□ 予算は正しい

□ 単価は正しい

□ 希望数は正しい

□ 予算チェック・自動計算を実施した

□ 手動調整した場合は再計算を実施した

□ エラー欄が「なし」になっている

□ 発注金額が予算内に収まっている

□ 発注結果を確認した

以上で操作完了です。

13. よくある困りごと（Q&A）
Q1. 「始める」を押したら入力した内容が消えました
原因

「始める」は初期化処理です。

押すと、

基本設定シート
発注数計算シート

の内容が削除されます。

対処方法

入力途中や計算後は押さないでください。

初回利用時や設定を最初からやり直したい場合のみ押してください。

Q2. 「発注数計算表作成」ができません
原因

基本設定シートの商品名または課名が入力されていません。

対処方法

基本設定シートを開き、

商品名
課名

を1件以上入力してください。

Q3. 「商品名が重複しています」と表示されました
原因

同じ商品名が複数入力されています。

例
商品名
コピー用紙
ボールペン
コピー用紙
対処方法

重複している商品名を修正してください。

Q4. 「途中に空欄があります」と表示されました
原因

商品名または課名の入力順が飛んでいます。

NG例
商品名
コピー用紙
（空欄）
ボールペン
対処方法

上から順番に詰めて入力してください。

Q8. 「予算内で増やしますか？」とは何ですか？
内容

希望数のままでも予算内に収まる場合に表示されます。

「はい」

予算を使い切る方向で数量を増やします。

「いいえ」

希望数のまま発注します。

Q9. 発注数が希望数より少なくなりました
原因

希望金額が予算を超えています。

対処方法

正常な動作です。

予算内に収まるよう自動調整されています。

Q10. 黄色セルは何ですか？
内容

手動調整できるセルです。

変更後は、

「手動調整・再計算」

を押してください。

Q11. 赤字になりました
原因

手動変更したセルです。

または、

その変更によって再計算されたセルです。

対処方法

問題ありません。

次回変更時に黒字へ戻ります。

Q12. 発注数合計を変更したら課別数量が変わりました
原因

仕様です。

発注数合計を変更すると、

希望数比率で課別数量を再配分します。

例

変更前

総務	営業	技術	合計
10	20	30	60

合計を72へ変更

↓

総務	営業	技術	合計
12	24	36	72
Q13. 課別数量を変更したら発注数合計が変わりました
原因

仕様です。

課別数量の合計を再計算しています。

Q14. 「変更された発注数セルがありません」と表示されました
原因

手動変更をしていない状態で

「手動調整・再計算」

を押しています。

対処方法

発注数表を変更してから実行してください。

Q15. エラー欄に「発注金額が予算を超過しています」と表示されました
原因

手動調整により予算超過しています。

対処方法

発注数を減らしてください。

操作

Q19. エラー欄が「なし」なのに結果がおかしい気がします
確認項目
商品名は正しいか
課名は正しいか
単価は正しいか
希望数は正しいか
予算は正しいか
手動調整後に再計算したか

を確認してください。

多いミス

「始める」を押してデータを消してしまう
対策

作業終了前に、

エラー欄が「なし」
差額が想定どおり

になっていることを確認してください。
